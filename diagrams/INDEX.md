CRISP-DM / CI-CD diagram was generated from PlantUML source kept under
`diagrams/src/`. Every chart was generated by `diagrams/charts/generate_charts.py`
through a one-shot Docker run. To regenerate:

```bash
# UML / architecture / methodology (35 diagrams)
cd diagrams/src && plantuml -tpng -o ../ *.puml

# Matplotlib charts (8 figures)
docker run --rm -v "$PWD":/work -w /work python:3.11-slim sh -c \
  "pip install --quiet matplotlib numpy && python diagrams/charts/generate_charts.py"
```
